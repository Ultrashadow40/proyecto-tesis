Uso del Computador

Encender el computador

  1. Presiona el botón de encendido.
  2. Espera a que cargue.
  3. Ingresa tu contraseña si la pide.

Usar el mouse

  1. Mueve el mouse para mover la flecha.
  2. Haz clic izquierdo para seleccionar.
  3. Haz doble clic para abrir.

Usar el teclado

  1. Usa las letras para escribir.
  2. Enter sirve para aceptar.
  3. Espacio separa palabras.

Navegar en internet

  1. Abre un navegador (Chrome, Edge).
  2. Escribe lo que buscas.
  3. Haz clic en un resultado.
