Programas del Computador

Microsoft Word

Sirve para escribir documentos

  1. Abrir Word
  2. Presionar “Nuevo documento”
  3. Escribir con el teclado
  4. Guardar el archivo

Navegador (Chrome, Edge)

Sirve para usar internet

  1. Abrir el navegador
  2. Escribir lo que buscas
  3. Presionar Enter
  4. Hacer clic en un resultado

Explorador de archivos

Sirve para ver tus archivos

  1. Abrir “Este equipo” o “Archivos”
  2. Buscar carpetas
  3. Hacer doble clic para abrir

Reproductor de música o video

Sirve para escuchar música o ver videos

  1. Abrir el programa
  2. Seleccionar archivo
  3. Presionar reproducir ▶️
